## Core Proof

Most systems prove that a decision was returned.

This engine proves whether a proposed continuation has enough lawful support to proceed.

```text
The proof is not that the system makes a decision.

The proof is that inadmissible consequence cannot continue, bind, or leave effect behind.
```

## Public Boundary Rule

```text
No continuation, action, or consequence may proceed unless it remains admissible under current state, capacity, burden, and effect safety.
```

---

## Outcome Model

| Outcome | Meaning | Effect posture |
|---|---|---|
| `CONTINUE` | Continuation remains admissible | May proceed |
| `PAUSE` | Near-boundary strain requires hold | No binding effect |
| `REBOUND` | Return to last stable support point | No new effect |
| `REFUSE` | Proposed consequence lacks standing | No effect |
| `HALT` | Integrity/capacity collapse | Corridor stopped |
